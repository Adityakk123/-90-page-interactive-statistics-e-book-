# Interactive Statistics E-Book

**Research & Instructional Design Project | Quarto, R | 2025**

## Overview
Designed and evaluated a 90-page interactive statistics e-book using Quarto,
built to improve learning retention among undergraduate students. Combined
statistical modeling, literature synthesis, and experimental design across
a behavioral learning study with 200+ participants.

## Key Outcomes
| Metric | Result |
|--------|--------|
| E-Book Length | 90 pages |
| Study Participants | 200+ students |
| Learning Retention Improvement | 40% (pre–post assessment analysis) |
| Format | Interactive Quarto document |

## What This Project Involved
1. **Literature Synthesis** — Reviewed existing research on statistics pedagogy
2. **Experimental Design** — Designed pre–post assessment framework for 200+ participants
3. **Content Development** — Built 90-page interactive e-book in Quarto
4. **Statistical Modeling** — Analyzed assessment data to measure retention outcomes
5. **Evaluation** — Validated 40% improvement in learning retention

## Skills Demonstrated
- Instructional design and structured content delivery
- Experimental design and behavioral research methodology
- Statistical modeling and quantitative evaluation
- End-to-end independent project management
